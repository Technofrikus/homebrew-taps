# Homebrew Taps

This is my personal Homebrew tap for various macOS applications and command-line tools.

## Included Casks

- **Pipette Desktop**: An independent, open-source keymap editor for Vial-powered keyboards.

## How to use

To add this tap to your Homebrew installation, run:

```bash
brew tap technofrikus/taps
```

Then you can install the software:

```bash
brew install --cask pipette
```

Alternatively, you can install directly without tapping first:

```bash
brew install --cask technofrikus/taps/pipette
```
